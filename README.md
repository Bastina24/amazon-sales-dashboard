# 📊 Amazon Sales Performance Dashboard

## 📌 Project Overview
This project focuses on analyzing Amazon sales data to understand revenue, profit, customer behavior, and product performance.  
An interactive dashboard was created using Looker Studio to visualize key business insights.

---

## 🛠️ Tools & Technologies
- Google Looker Studio  
- Microsoft Excel / Google Sheets  

---

## 📁 Dataset
The dataset is available in the `data` folder.

### Columns included:
- order_id  
- order_date  
- product_id  
- product_category  
- price  
- discount_percent  
- quantity_sold  
- customer_region  
- payment_method  
- rating  
- review_count  
- discounted_price  
- total_revenue  
- profit

- Top performing categories: **Electronics & Fashion**  
- Regional sales distribution analyzed
